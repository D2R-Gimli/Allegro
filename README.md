# Unofficial Allegro Versions Collection

**Disclaimer:** This repository is an **unofficial** collection of Allegro library versions. It is intended **solely for historical reference and development purposes**, particularly for compiling old game tools and projects. This repository is **not affiliated with or endorsed by the official Allegro project**.

---

## Overview

Allegro is a cross-platform library widely used for game development and multimedia applications. Many older projects rely on previous versions of Allegro, which can be hard to find today. This repository collects several historical Allegro releases to make it easier for developers to compile legacy tools or games.

---
